# Full Stack Online Food Delivery Application

Building a **Full Stack Online Food Delivery Application** using **React + Spring Boot + MongoDB** involves three major parts:

1. Customer Web Application (React)  
2. Admin Panel (React)  
3. Backend REST API (Spring Boot + MongoDB)
---

# 🛠 Technologies Used

## Frontend
- React
- Vite
- Tailwind CSS
- Axios
- React Router
- Context API / Redux

## Backend
- Spring Boot
- Spring Security
- JWT Authentication
- MongoDB
- Spring Data MongoDB
- Maven

---

#  Major Features

##  Authentication

- User Registration
- User Login
- JWT Based Authentication
- Secure API Requests

## 🍕 Browse Food Menu

Users can:

- View restaurant menu
- View food categories
- View food details

## 🛒 Cart System

- Add food items to cart
- Remove items from cart
- Update quantity
- View total price


## Application Workflow

1. User opens the website  
2. Browse food items  
3. Add items to cart  
4. Checkout  
5. Order stored in MongoDB  
6. Admin receives order  
7. Admin updates order status  
8. User tracks order
